# Client Lead Management System (Mini CRM)

## Overview

This project is a simple Client Lead Management System (Mini CRM) developed as part of an internship task. The application allows businesses to collect leads from a website contact form, store them in a database, and manage them through an admin dashboard.

The system simulates how real businesses track potential clients and convert them into customers.

## Features

* Add new leads through a contact form
* Store lead information in the database
* View all leads in a dashboard
* Track lead status (New, Contacted, Converted)
* Backend APIs for managing leads
* Simple and clean user interface

## How to Run the Project

1. Install Node.js on your system.
2. Clone this repository.
3. Install dependencies:

npm install

4. Start the server:

node server.js

5. Open index.html to submit new leads.
6. Open dashboard.html to view and manage leads.

## Learning Outcome

This project helped in understanding full-stack development by connecting frontend forms, backend APIs, and database systems. It demonstrates how data flows from a user interface to a backend server and gets stored in a database.
